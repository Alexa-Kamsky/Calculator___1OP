# Calculator___1-OP
дз 1 ОП

Здесь предствален код на Python, который представляет собой программу-парсер арифметических выражений.

Что это значит? Эта программа:
- Считает из командной строки выражение. Например: (2 + 3) * 4 - 10 / 2.
- Корректно вычисляет результат этого выражения, учитывая приоритет операторов: скобки, затем умножение и деление, затем сложение и вычитание.
- Выводит результат.

Строки-выражения разбираются и выражение вычисляется само.

Задача решалась через:

- Токенизацию. Разбитие входной строки "(2 + 3)*4" на отдельные элементы – токены. Т.е. получаем список: ['(', '2', '+', '3', ')', '*', '4']. Проходим по строке символ за символом. Если символ – цифра, вы начинаете читать число целиком. Если символ – оператор +, -, *, / или скобка (, ), – это отдельный токен. Пробелы должны игнорироваться
- Приведение исходной строки к обратной польской нотации (RPN), используя алгоритм сортировочной станции.
- Вычисление RPN.

Чтобы протестировать работу программы выполните следующее:
1. Склонируйте репозиторий (https://github.com/Alexa-Kamsky/Calculator___1-OP.git)
2. Перейдите в папку с проектом (Calculator___1-OP)
3. Запустите программу (py Калькулятор.py)
4. Введите свое выражение (например, 567+2*102-44/4 //ответ: 760)
5. Программа завершит свою работу. Если хотите протестировать еще, вернитесь к шагу 3.
